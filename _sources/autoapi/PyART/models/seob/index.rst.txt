.. py:function:: CreateDict(M=1.0, q=1, chi1z=0.0, chi2z=0, chi1x=0.0, chi2x=0, chi1y=0.0, chi2y=0, dist=100.0, iota=0, f0=0.0035, df=1.0 / 128.0, dt=1.0 / 2048, phi_ref=0.0, e0=0.0, rel_anomaly=np.pi, use_geom='yes', approx='SEOBNRv5HM', use_mode_lm=[(2, 2)])

   Create the dictionary of parameters for pyseobnr->GenerateWaveform

   :param M: Total mass in solar masses. Default is 1.0.
   :type M: float
   :param q: Mass ratio m1/m2 >= 1. Default is 1.
   :type q: float
   :param chi1z: Dimensionless spin of the primary along the orbital angular momentum.
                 Default is 0.0.
   :type chi1z: float
   :param chi2z: Dimensionless spin of the secondary along the orbital angular momentum.
                 Default is 0.0.
   :type chi2z: float
   :param chi1x: Dimensionless spin of the primary in the orbital plane (x-component).
                 Default is 0.0.
   :type chi1x: float
   :param chi2x: Dimensionless spin of the secondary in the orbital plane (x-component).
                 Default is 0.0.
   :type chi2x: float
   :param chi1y: Dimensionless spin of the primary in the orbital plane (y-component).
                 Default is 0.0.
   :type chi1y: float
   :param chi2y: Dimensionless spin of the secondary in the orbital plane (y-component).
                 Default is 0.0.
   :type chi2y: float
   :param dist: Distance to the source in Mpc. Default is 100.0.
   :type dist: float
   :param iota: Inclination angle in radians. Default is 0 (face-on).
   :type iota: float
   :param f0: Starting frequency of the (2,2) mode in geometric units. Default is 0.0035.
   :type f0: float
   :param df: Frequency resolution in Hz. Default is 1/128.
   :type df: float
   :param dt: Time step in seconds. Default is 1/2048.
   :type dt: float
   :param phi_ref: Reference phase at f0 in radians. Default is 0.0.
   :type phi_ref: float
   :param e0: Initial eccentricity at f0. Default is 0.0.
   :type e0: float
   :param rel_anomaly: Relativistic anomaly at f0 in radians. Default is pi.
   :type rel_anomaly: float
   :param use_geom: Whether to use geometric units ('yes' or 'no'). Default is 'yes'.
   :type use_geom: str
   :param approx: Approximant to use ('SEOBNRv5HM', 'SEOBNRv5PHM', 'SEOBNRv5EHM').
                  Default is 'SEOBNRv5HM'.
   :type approx: str
   :param use_mode_lm: List of (l, m) tuples specifying which modes to use. Default is [(2, 2)].
   :type use_mode_lm: list of tuple
