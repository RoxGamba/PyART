   .. py:method:: get_pn_quantity(name=None, path=None, order=None, variable='x')

      Return a catalog quantity as an analytic expression wrapper.

      When ``order`` is provided, the returned quantity is truncated
      relative to the leading power of ``variable``.

      :param name: Indexed quantity name used for lookup.
      :type name: str or None, optional
      :param path: Direct path to the Mathematica file.
      :type path: str or None, optional
      :param order: PN order relative to the leading power of ``variable``.
      :type order: object, optional
      :param variable: Variable used for PN order counting. When the selected variable
                       is not present and the quantity has exactly one variable, that
                       variable is inferred automatically.
      :type variable: str or sympy.Symbol, optional

      :returns: Parsed BHPT series wrapped for symbolic and numerical use.
      :rtype: AnalyticExpression
