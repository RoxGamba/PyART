.. py:class:: BHPTEntry

   Parsed representation of a single BHPT series file.

   :param key: Canonical catalog key for the entry.
   :type key: str
   :param path: Absolute path to the Mathematica source file.
   :type path: str
   :param metadata: Metadata association extracted from the file.
   :type metadata: dict[str, Any]
   :param assignments: Top-level Mathematica assignments found in the source.
   :type assignments: dict[str, str]
   :param definitions: Delayed Mathematica definitions found in the source.
   :type definitions: dict[str, str]
   :param expr: Parsed SymPy expression for the selected series.
   :type expr: sympy.Basic
   :param quantity: Wrapped analytic expression corresponding to ``expr``.
   :type quantity: AnalyticExpression

   .. py:method:: get_entry(name=None, path=None)

      Resolve, parse, and cache a BHPT Mathematica file.

      The returned :class:`BHPTEntry` contains the resolved key and path,
      parsed
      metadata, raw assignment tables extracted from the Mathematica
      source, the SymPy expression, and the corresponding
      `AnalyticExpression` wrapper.

      :param name: Indexed quantity name used for lookup.
      :type name: str or None, optional
      :param path: Direct path to a Mathematica file inside the indexed tree.
      :type path: str or None, optional

      :returns: Parsed and cached representation of the requested series file.
      :rtype: BHPTEntry

      :raises ValueError: If the source or selected expression cannot be parsed.



   .. py:method:: _resolve_entry(name=None, path=None)

      Resolve a user query to an indexed BHPT file path and canonical key.

      A direct path lookup wins when `path` is provided, but the resolved
      file must still live inside the configured BHPT tree. Otherwise the
      method performs a normalized token match on the indexed quantity keys.

      :param name: Indexed quantity name used for lookup.
      :type name: str or None, optional
      :param path: Direct path to a Mathematica file inside the indexed tree.
      :type path: str or None, optional

      :returns: Canonical quantity key and absolute file path.
      :rtype: tuple[str, str]
